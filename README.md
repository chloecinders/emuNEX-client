# emuNEX (client)
Remote Emulation & Library Management.

![image](https://github.com/user-attachments/assets/cf2cb2fe-ffd4-4c2a-8338-d1ccf58fba0c)

emuNEX is a service, think of it like Steam, but for emulators. The client connects to a remote server which provides emulators, games and in the future could handle features such as profiles/friends.

Playing old games is as easy as searching for a game and pressing (download and then) play!

Find the source code for the server here: https://github.com/chloecinders/emuNEX-server

## (Legal) Disclaimer

This client does not work without an emuNEX server. This client itself provides no emulator or game downloads. Games and emulators which are downloaded with the help of this client are subject to their respective copyright holders. emuNEX does not support any form of piracy.

## Installation

Grab an installer from https://github.com/chloecinders/emuNEX-client/releases!

## Configuring

Configuring emulators can be done within the client in the emulators section.
As different emulators work differently and have different ways of running and storing their save files we keep track of which configurations work for emulators in the [configurations](CONFIGURATIONS.md) file. If you confirm an emulator works with a configuration please add it to the file!
